Guardian
========

Guardian is originally created for XHTML and CSS editing, however you can use it
for Ruby and JavaScript development as well.

Announcement
------------

Fork from http://www.vim.org/scripts/script.php?script_id=1240, just
for managing by [Vundle](https://github.com/gmarik/vundle)

Install
-------

With [Vundle](https://github.com/gmarik/vundle)

    Bundle 'Suave/vim-colors-guardian'
